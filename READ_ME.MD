# AI-Driven Retail Demand & Customer Insight System

## Objective
Analyze customer behavior, segment users, and forecast monthly demand using Python.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn

## Key Features
- Synthetic business data generation
- Customer segmentation using K-Means
- Demand forecasting using Linear Regression
- Automated business insights

## Business Impact
- Identified high-value customers
- Found best segments to target
- Improved demand planning decisions

## Outcome
Created actionable insights for retail growth and marketing strategy.
